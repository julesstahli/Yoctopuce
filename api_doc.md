Yoctopuce api
=============

Measures
--------

### /api/measures

Get multiple measures
```
Javascript using Axios
```
```javascript
axios.get([endpoint]/api/measures?limit=50&from=2020-01-13 00:00:00&to=2020-02-13 00:00:00).then(data => {
  console.log(data);
}, error => {
  console.error(error);
});
```

Params

| Name | Value | Default |
| --- | --- | --- |
| limit | integer | null |
| from | datetime | null |
| to | datetime | null |
